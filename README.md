# Gaia Mnemosyne

Universal Knowledge Engine (RAG + Graph + Ontology)
- Rust backend + React UI
- Vector search + Graph DB + Providers architecture

## Development

```bash
make bootstrap
make up
```

- Requires Rust toolchain: 1.91.1
- To install:
```bash
rustup toolchain install 1.91.1
rustup default 1.91.1
```
